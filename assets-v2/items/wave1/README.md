# Firestaff V2 Items / Action Icons Wave 1

## Purpose

This directory is the bounded V2 hook for item sprites, inventory icons, and action-hand symbols.

## Intended first families
- standard inventory item icon set
- action-hand glyph set
- empty-hand glyph
- small HUD object markers

## Style contract
- Silhouette and gameplay readability come first.
- Keep icons chunky, painted, and limited in palette separation.
- Avoid glossy heroic rendering that collapses at HUD scale.

## Honest current state
- No production assets are checked in here yet.
- This directory is a class brief and next-step hook only.
