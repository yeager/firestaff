# Firestaff V2 Typography / Text Wave 1

## Purpose

This directory is the bounded V2 hook for HUD labels, action text, spell text, and menu-facing text-bearing surfaces.

## Intended first families
- HUD labels
- action names
- spell text payloads
- small state markers

## Style contract
- Replace old bitmap text with a restrained V2 text system.
- Keep the tone carved, archival, or utilitarian rather than sleek-app modern.
- Text should support structure and readability, not become decorative noise.

## Honest current state
- No V2 typography assets are checked in here yet.
- This directory is a class brief and next-step hook only.
