# V2 Wave 1 Spec: Status Boxes

## Scope
- Left and right status frame shells
- No portraits, no labels, no stat typography

## Canonical sizes
- 4K master: **670×290** each
- 1080p derived: **335×145** each

## Variants in this family
- Left status frame
- Right status frame

## Art direction stub
- Painterly dark-fantasy shell with crisp inner readability
- Surfaces can suggest metal, carved stone, lacquered wood, or mixed materials
- Must support later portrait and status overlays cleanly

## Guardrails
- Preserve the classic silhouette
- Avoid decorative elements that crowd portrait or text regions
- Left/right versions should feel like a designed pair, not a mirrored shortcut unless that proves visually correct
