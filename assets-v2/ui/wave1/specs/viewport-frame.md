# V2 Wave 1 Spec: Viewport Frame

## Scope
- Core gameplay frame shell
- Viewport base/frame
- No world art, no environment paint, no final scene rendering

## Canonical sizes
- Core gameplay frame master: **3200×2000**
- Core gameplay frame derived: **1600×1000**
- Viewport base master: **2240×1360**
- Viewport base derived: **1120×680**

## Art direction stub
- Dark fantasy painterly UI frame
- Heavy but readable silhouette
- Worn stone/bronze/iron hybrid acceptable
- Interior edge must support dynamic world rendering without muddying the viewport

## Production requirements
- Preserve DM1 proportion exactly
- Avoid photorealism
- Avoid fake pixel-upscale texture
- Keep the frame modular and repaint-friendly

## Deliverables later
- Base frame paint
- Clean mask/alpha export
- Optional ornament variant set
