# Firestaff V2 hybrid hero portrait PoC — iconic set

Follow-up to the Halk portrait proposal. This set creates first-pass hybrid portraits for four more iconic Dungeon Master champions:

- Alex Ander (#6)
- Nabi The Prophet (#7)
- Sonja She Devil (#10)
- Syra Child Of Nature (#21)

Each has:

- source crop under `../source/`
- generated candidate in this directory (`1024×1536`)
- provisional 4K-ish master under `masters-4k/` (`2160×3240`)

These are style proposals, not final production art. The main test is whether the original tiny portrait remains recognizable after modern rendering.
