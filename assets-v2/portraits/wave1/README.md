# Firestaff V2 Portraits Wave 1

## Purpose

This directory is the bounded V2 hook for champion portrait presentation.

## Intended first families
- champion portrait busts
- portrait presentation frames

## Style contract
- Portraits should read as compact HUD payloads, not splash-art posters.
- Favor face planes, equipment silhouette, and readable value grouping.
- Keep backgrounds quiet enough for text and bars to remain legible nearby.
