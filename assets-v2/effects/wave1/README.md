# Firestaff V2 Effects / Magic Wave 1

## Purpose

This directory is the bounded V2 hook for projectile, impact, cast, glow, and trail families.

## Intended first families
- projectile sprites
- spell impact layers
- cast / trail overlays
- bounded ambient mist or poison overlays where gameplay needs them

## Style contract
- Effects must stay subordinate to gameplay geometry.
- Use layered clarity, not bloom-heavy spectacle.
- Preserve classic DM readability and timing feel.

## Honest current state
- No production assets are checked in here yet.
- This directory is a class brief and next-step hook only.
