# Firestaff V2 Environment / Structural Wave 1

## Purpose

This directory is the bounded V2 hook for environment-facing asset families that are already discoverable in repo planning, but do not yet have a checked-in generation pack.

## Intended first families
- door frames
- door side graphics
- stairs
- wall-set structural modules
- floor / ceiling base planes
- wall / floor ornaments

## Style contract
- Preserve classic DM depth grammar and readable large planes.
- Keep forms carved and weighty rather than glossy or cinematic.
- Build modular families before attempting broad scenic repaint coverage.

## Honest current state
- No production assets are checked in here yet.
- This directory is a class brief and next-step hook only.
