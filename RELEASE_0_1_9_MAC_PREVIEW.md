# Firestaff 0.1.9, macOS Preview

Firestaff 0.1.9 is an early macOS preview release.

This preview includes continued work on:
- M11 V1 parity and runtime/presentation convergence toward ReDMCSB
- corrected in-game DM PC VGA palette path and follow-up visual tuning groundwork
- ownership/parity analysis for movement, doors, environment, and broader 1:1 planning
- rebuilt V2 UI asset families from corrected original reference/mapping assumptions
- trusted-scope original-vs-V2 comparison material and asset-reference rules

What this is:
- a preview build for testing current macOS progress
- a checkpoint for ongoing DM1/V1 parity work
- an honest snapshot of current Firestaff runtime and packaging state

What this is not:
- not full Dungeon Master parity yet
- not a finished cross-platform release
- not a claim that V1 is visually complete
- not a claim that V2 visuals are final or accepted wholesale

Known status:
- V1 palette/path correctness is materially better than older previews, but final call-site color tuning still remains
- ownership migration from custom M11 glue toward source-faithful runtime paths is still ongoing
- V2 UI assets have been rebuilt more conservatively against verified references, but broader V2 work is still exploratory and bounded
- comparison material is now split into trusted and invalidated scopes; older broad PDF comparisons should not be treated as authoritative
