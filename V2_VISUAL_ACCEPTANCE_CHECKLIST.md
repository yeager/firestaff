# Firestaff V2 Visual Acceptance Checklist

Use this checklist before approving any new V2 in-game image pass.

## ReDMCSB-first gate
- [ ] The image was reviewed against ReDMCSB / DM1 in-game composition, not against generic fantasy UI taste.
- [ ] Any visible departure from ReDMCSB has a clear, defensible V2 reason.
- [ ] If no strong reason exists, the Firestaff variant was treated as wrong.

## Recognizability gate
- [ ] At first glance, the image reads as Dungeon Master.
- [ ] At thumbnail size, it still reads as a DM-style gameplay screen.
- [ ] It does not read first as a mockup, dashboard, poster, or promo board.

## Viewport gate
- [ ] The viewport is the dominant read.
- [ ] The dungeon scene inside the viewport is active and legible.
- [ ] The frame supports the scene instead of overpowering it.

## Depth gate
- [ ] Wall/floor/ceiling cues create clear first-person dungeon depth.
- [ ] The scene shows readable front/mid/far staging.
- [ ] The image does not feel flat, poster-like, or layer-stacked.

## Creature gate
- [ ] Any creature reads as an encounter inside the dungeon.
- [ ] Creature scale matches the scene's perspective.
- [ ] The creature does not look like a pasted-in hero illustration.

## UI gate
- [ ] The right-side structure still feels DM-like, not like a modern control sidebar.
- [ ] The bottom HUD reads as compact game structure, not a dock.
- [ ] Empty decorative UI mass was removed or reduced.
- [ ] No element reads as a generic app widget.

## Palette / material gate
- [ ] The palette stays restrained and DM-readable.
- [ ] No synthetic cyan/teal accent blocks or similar modern scaffolding remain.
- [ ] Material treatment feels closer to aged frame / stone / bitmap-display logic than glossy fantasy UX.

## Hard reject triggers
Reject the image immediately if any of these are true:
- [ ] viewport is not the main event
- [ ] creature reads like promo art instead of encounter staging
- [ ] UI reads like a modern dashboard
- [ ] composition reads like a presentation board
- [ ] palette/material treatment breaks DM recognizability

## Approval rule
Approve a V2 in-game image only if every gate above passes.
