# Firestaff 0.3.0 Release

## Overview
This release includes:

- Updated version to 0.3.0
- Various bug fixes and improvements
- Performance enhancements

## Changes
- Version bump from 0.11.0 to 0.3.0
- Updated build system configurations
- Improved asset handling and version management

## Installation
To build Firestaff 0.3.0:
```bash
mkdir build
cd build
cmake ..
make
```

## Notes
This release focuses on stability improvements and preparing for upcoming features.
