# Firestaff 0.3.0 Release Announcement

We're pleased to announce the release of Firestaff 0.3.0, an important update that brings significant improvements to the project.

## Key Changes

- **Version Bump**: Updated from 0.11.0 to 0.3.0 
- **Build System**: Enhanced CMake configuration
- **Stability**: Various bug fixes and stability improvements

## Getting Started

To build and run Firestaff 0.3.0:
```bash
mkdir build
cd build
cmake ..
make
./firestaff
```
