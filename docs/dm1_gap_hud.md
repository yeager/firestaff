# GAP: HUD — Top Status Strip Does Not Exist in ReDMCSB DM1 V1

## Status
**GAP — No implementation needed; DM1 V1 has no top-status-bar; this is a DM2/CSB feature**

## What the Top HUD Strip Is

In DM2 and CSB (Dungeon Master II, Chaos Strikes Back), the top of the screen displays a persistent status strip showing:
- **Dungeon name** — the name of the current dungeon/level
- **Floor indicator** — current dungeon floor (e.g., "FLOOR 3")
- **Game clock** — elapsed time since game start
- **Gold count** — partys current gold
