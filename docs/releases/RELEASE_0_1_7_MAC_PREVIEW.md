# Firestaff 0.1.7, macOS Preview

Firestaff 0.1.7 is an early macOS preview release.

This preview includes continued work on:
- M11 V1 graphics and parity groundwork
- M12 modern launcher input and usability improvements
- mouse hit-testing in the launcher
- immediate language switching in the launcher
- improved launcher navigation and visual polish

What this is:
- a preview build for testing progress on macOS
- a checkpoint for launcher and runtime work

What this is not:
- not full Dungeon Master parity yet
- not a finished cross-platform release
- not a claim that the in-game V1 graphics are done

Known status:
- the modern launcher is improved and more usable
- the in-game DM1 presentation still needs major graphics/parity work before it feels recognizably like classic Dungeon Master
