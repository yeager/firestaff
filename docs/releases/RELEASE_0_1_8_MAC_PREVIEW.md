# Firestaff 0.1.8, macOS Preview

Firestaff 0.1.8 is an early macOS preview release.

This preview includes continued work on:
- M11 V1 parity and presentation improvements
- corrected in-game DM PC VGA palette path for the V1 screen
- projectile travel, detonation, explosion aftermath, and DM1 bitmap-backed explosion rendering
- bounded V2 groundwork, assets, and experimental integration

What this is:
- a preview build for testing current macOS progress
- a checkpoint for M11 parity work and ongoing V2 experimentation
- an honest snapshot of the current runtime

What this is not:
- not full Dungeon Master parity yet
- not a finished cross-platform release
- not a claim that V1 is visually complete
- not a claim that current V2 visuals are accepted direction

Known status:
- V1 color character is significantly improved over earlier previews
- M11 action/projectile/explosion behavior is substantially more Dungeon Master-like than before
- movement/door/environment ownership still needs further convergence toward ReDMCSB-faithful runtime ownership
- V2 remains exploratory and is being reset against ReDMCSB before further broad art production
