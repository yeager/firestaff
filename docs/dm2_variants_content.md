# DM2 V1 — Content Differences Between Releases

## Patch Levels
- v1.0: Initial release (DOS, 1994)
- v1.1: Bug fix patch (if any existed)
- No major content patches known

## Differences from DM1
- New storyline: Chaotic Kingdom, Soul of a Robot quest
- New enemy types and dungeon aesthetic
- 17 dungeons vs DM1 dungeon structure
- 4 difficulty tiers: Adventurer/Journeyman/Expert/Novice rating system
- New spell system with different magic types
- Dungeons can collapse/change while exploring

## Version-specific content changes
None identified between DOS and Windows releases - same game binaries with
Windows executable replacing DOS one.

## Data files
- SKULL.ASM: Main game file (DOS)
- No variant data files like DM1 had (no separate amiga/atari data)
- All language content embedded in single executable

## Conclusion
DM2 content was stable across releases. No content-variant files found like
DM1 multilingual GRAPHICS.DAT differences.
