# DM2 V1 — Language Variants

## Question: Does DM2 have multilingual releases like DM1?

Based on research, DM2 (Dungeon Master II: The Legend of Skullkeep) was released primarily in English.
- Original releases: DOS (1994), Windows 3.1 (1995)
- No known Japanese-language release like DM1 CSB had
- European markets likely got English versions (possibly with localized manuals)
- No evidence of localized game binaries (no Japanese DAT files like DM1 had)

## Evidence from skproject (source):
- Source code comments in English
- No language-switching code visible
- String tables in English only

## Conclusion
Unlike DM1 CSB which had full Japanese translation (j-TITLE, j-GRAPHICS, j-DUNGEON), DM2 V1 appears to be English-only at the binary level. Manual/localization may have existed regionally but no variant game files found.

SHA256 reference (DOS EN release): Dungeon-Master-II-Skullkeep_DOS_EN.zip
