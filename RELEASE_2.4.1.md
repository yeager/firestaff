# Firestaff v2.4.1

DM1 V1/V2 parity and platform verification release.

## Highlights

- Source-locks DM1 V2 lighting palette selection against the ReDMCSB torch and magical-light thresholds.
- Adds bounded gates for creature projectile attack payloads, silent spell actions, mouth consume animation, inert potion consumption, direct audio marker buckets, and poison damage-over-time behavior.
- Improves touch and startup behavior, including the centered startup-menu version footer.
- Verifies the Linux build path and keeps the platform release flow on GitHub Actions.

## Platform Artifacts

GitHub Actions builds and attaches:

- macOS DMG and app ZIP
- Windows ZIP and installer EXE
- Linux DEB and RPM packages
- SHA256 checksum files

## Notes

Original Dungeon Master game data is still required and is not included.
